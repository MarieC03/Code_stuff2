Code of Conduct
=================


Our goal is to maintain a professional, collaborative, and respectful environment for all contributors and users.

Respectful Conduct
----------------------------

- Treat all contributors and users with respect and professionalism.
- Be considerate in discussions, reviews, and issue reports.
- Disagreements happen; handle them constructively and focus on facts, not personalities.

Contributions
------------------------

- Contributions are welcome via pull requests or issues.
- There is no expectation of authorship in scientific publications solely from contributing code, documentation, or bug fixes.

Reporting
--------------------

- If you observe behavior that violates this code of conduct, please contact the project owner directly via email.
- Reports will be handled confidentially and respectfully.

Attribution
-----------

- Users of GRACE are encouraged to cite the project in publications where the software is used.
- See the project's main documentation for the recommended citation format.

Acknowledgements
----------------

We welcome contributions and discussions from all backgrounds. Thank you for helping make GRACE a robust and collaborative project.
